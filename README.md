Features
CGPA Planner App
descripton
•A CGPA planner app that helps students plan and achieve their academic goals by calculating required grades based on current performance and upcoming courses. It also allows grade simulation to visualize possible outcomes and make better decisions.

•Academic Input • Allows students to enter their current CGPA, completed courses, grades, and ECTS credits to build a foundation for accurate calculations.
•Course Planning • Enables users to add upcoming courses and define a target GPA or final CGPA they want to achieve in future semesters.
•Grade Calculator • Automatically calculates the required grades in each course needed to reach the user’s target CGPA or semester GPA.
•Grade Simulation • Lets users adjust expected grades for planned courses and instantly see the minimum and maximum CGPA outcomes to support better academic decisions
