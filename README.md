FarmKeeper is a local-only mobile app designed for farmers to manage their crops, track watering schedules, and monitor harvest dates. Farmers can register each crop they plant (name, type, planting date, and days to maturity), and the app automatically calculates the expected harvest date. The app sends push reminders for watering based on custom intervals and notifies the farmer when harvest time is approaching.

Features

1. Crop Registration
Add new crops with name, type (Maize, Rice, Tomato, etc.), variety, planting date, days to maturity (auto-calculates harvest date), field/plot name, quantity planted, and optional photo. Edit or delete crops anytime. View all active crops separately from harvested history. Search by name or field, filter by type or status, and sort by planting date or harvest date.

2. Watering Reminders
Set custom watering intervals per crop (daily, every 2/3/7 days, or weekly). Mark crops as watered with one tap, view last watered date and days since last watering. Receive push notifications when watering is due and for overdue watering. 

3. Harvest Tracking
Auto-calculates harvest date from planting date + maturity days. Displays harvest countdown timer. Sends push notifications at 7 days before, 3 days before, on harvest day, and for overdue harvest. When harvesting, record actual harvest date, yield amount with unit (kg, bags, tons), quality rating, and harvest notes. Compare estimated vs actual harvest dates and yields.

4. Crop Journal
Add daily text notes with optional photo attachments. Categorize notes as Weather, Pest, Disease, Fertilizer, Irrigation, or General. For weather notes, select condition (Sunny, Rainy, Cloudy, Windy, Storm). For pest/disease notes, record pest name and treatment applied. For fertilizer notes, record fertilizer name and quantity. View all notes as a timeline, filter by category, edit or delete any note, and search notes by keyword.

5. Dashboard & Statistics
Overview screen showing total active crops, total harvested crops (all time and this season), and total fields/plots. Lists upcoming watering reminders for next 7 days and upcoming harvests for next 14 days. Displays recent journal entries preview. Includes visual charts: bar chart comparing planted vs harvested this season, line chart showing monthly crop plantings, and pie chart showing crop type distribution. Shows statistics like most planted crop type and average yield per crop type.

6. Calendar View
Monthly calendar displaying all crop activities. Shows 💧 icons on watering due dates, 🌾 icons on harvest dates, and 🌱 icons on planting dates. Color-coded entries for quick identification (green for watering, orange for harvest). Tap any date to see all crops needing attention that specific day. Navigate between months with swipe gestures.

7. Local Storage, Export & Security
All data stored completely offline using SQLite – no internet permission required. Export any data to CSV (crops list, harvest history, or journal notes separately) and share via any app. Backup entire database to device storage and restore from backup file. Secure the app with optional PIN lock or biometric unlock (fingerprint/face ID). Global notification settings to enable/disable all notifications, toggle watering/harvest notifications separately. Dark mode and light mode toggle.
